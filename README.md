# TagoIO (tago-io)

TagoIO is an IoT cloud application development platform that enables businesses to build, deploy, and manage IoT applications with a comprehensive REST API. The platform provides full programmatic control over devices, dashboards, data storage, analysis scripts, alerts, and user management. TagoIO exposes regional REST API endpoints for device data ingestion, entity management, access control policies, and file storage, secured with token-based authentication. Plans range from a free tier supporting five devices up to an enterprise-grade TagoDeploy option supporting millions of devices.
